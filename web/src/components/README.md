# components

Reusable React components organised by feature domain. Each subfolder contains components specific to that domain.

## Domain Folders

| Folder | Purpose |
|---|---|
| `common/` | Shared UI components used across the application |
| `layout/` | Layout components (header, footer, sidebar, navigation shell) |
| `navigation/` | Navigation-specific components (breadcrumbs, menus) |
| `ageGroup/` | Age group display and management components |
| `coach/` | Coach profile and certification components |
| `formation/` | Formation builder and display components |
| `kit/` | Kit builder and order management components |
| `match/` | Individual match components (lineups, reports, events) |
| `matches/` | Match listing and scheduling components |
| `player/` | Individual player components (attributes, ratings, cards) |
| `players/` | Player listing and roster management components |
| `stats/` | Statistics and chart components (Recharts) |
| `tactics/` | Tactic editor and display components |
| `team/` | Team overview and management components |
| `training/` | Training session and drill components, including the drill builder canvas with players, cones, balls, markers, mannequins, and goals |
