# contexts

React Context providers for cross-cutting application state.

## Files

| File | Purpose |
|---|---|
| `AuthContext.tsx` | Authentication state — current user, login/logout, token refresh |
| `ThemeContext.tsx` | Theme state — club colours, dark/light mode |
| `NavigationContext.tsx` | Navigation state — sidebar open/close, active route |
| `PageTitleContext.tsx` | Page title state — dynamic document titles |
| `UserPreferencesContext.tsx` | User preferences — display settings, saved filters |
